# beetles-mobile

## Installation

```bash
yarn install
```

## Run

```bash
yarn android
```
or
```bash
yarn ios
```
or
```bash
yarn start
```

## Build
https://docs.expo.dev/build/setup/

```bash
npm install -g eas-cli
```

```bash
eas login
```

```bash
eas build --platform android
```

```bash
eas build --platform ios
```

```bash
eas build --platform all
```
